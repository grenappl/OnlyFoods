# OnlyFoods

A swipe-based food discovery mobile application that helps users explore meals and recipes in a fun, interactive way.

## Overview

OnlyFoods is a modern food discovery application inspired by swipe-based platforms. Instead of scrolling through long lists, users can quickly discover food ideas by swiping through visually appealing cards.

The app focuses on simplicity, personalization, and user experience, making it easier for users to decide what to eat or cook.

## Features

### Swipe-Based Food Discovery
- Swipe right to like a food item
- Swipe left to skip
- Interactive card-based interface

### Recipe Details View
- View full recipe information including:
    - Ingredients
    - Instructions
    - Description

### Recipe Creation
- Create and upload your own recipes
- Add ingredients and step-by-step instructions
- Edit or delete your recipes anytime

### Favorites System
- Save liked food items
- Access them anytime in your Favorites section

### Search and Filter
- Search food or recipes by name
- Filter by category (e.g., breakfast, lunch, dinner)

### User Profiles
- Create and manage your profile
- View your saved and created recipes

### Simple Recommendation System
- Suggests food based on user preferences and interactions
